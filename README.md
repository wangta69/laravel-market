# Laravel Market
## How to install laravel market

### 라라벨 인스톨 및 기본 처리
> 이 부분이 준비 되었으면 바로 다음 단계로 이동
1. laravel install
```
composer create-project laravel/laravel example-app
```
```
cd example-app
```
.env db 정보 세팅



### 라라벨 마켙 install
```
composer require wangta69/laravel_market
php artisan market:install
```
